# Video room
### Dependencies

- NodeJS Version 14

### Installing

Open Terminal, then type command:

Go to project folder :
``` 
> npm install
```

### Executing program
```
> npm start
> 
> open http://localhost:3000 in your browser with parameter chanelName in url
```

### Stack technology
- ReactJS (CRA)
- React-router-dom (Routing in React)
- Axios (Http Request)
- React Hooks
- Agora
- agora-rtc-react

### Features
- Vidio room
